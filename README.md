 # Search Engine Product Analytics using SQL

## Project Summary
This project analyzes product usage and revenue performance of a search engine platform using SQL.

## Dataset
The dataset contains structured records across:
- google_users
- searches
- clicks
- ads
- ad_clicks

## Tools & Technologies
- MySQL
- SQL

## Key Analysis Performed
- User Engagement Analysis  
- Search Trends & Click-Through Rate (CTR) Analysis  
- Advertisement Performance & Revenue Optimization  
- Business Growth Insights

## Business Impact
The analysis provides actionable insights to improve user engagement, optimize advertising strategies, and maximize revenue growth.
