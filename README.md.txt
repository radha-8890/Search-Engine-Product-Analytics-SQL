# Search Engine Product Analytics using SQL

## Project Summary
This project analyzes product usage and revenue performance of a search engine platform using SQL.

## Dataset
Contains 400K+ records across:
- google_users
- searches
- clicks
- ads
- ad_clicks

## Tools Used
MySQL, SQL

## Key Analysis
- User engagement metrics
- Search trend analysis & CTR
- Advertisement performance & revenue
- Business growth insights

## Outcome
Provided actionable insights to improve user engagement, optimize ad revenue, and drive platform growth.
